   TTS - TEXT TO SPEECH APP
===================================

✅ FILE NÀY AN TOÀN - KHÔNG CÓ VIRUS!

Ứng dụng được xây dựng bằng:
- Python (ngôn ngữ lập trình)
- PyInstaller (công cụ đóng gói)
- Flask (web framework)
- ElevenLabs API (text-to-speech)

CÁCH SỬ DỤNG:

1. Double click file "TTS-App.exe"

2. Nếu Windows hiển thị cảnh báo:
   → Click "More info"
   → Click "Run anyway"

   (Đây là cảnh báo tiêu chuẩn của Windows
    cho app chưa có chữ ký số)

3. Nếu Antivirus chặn:
   → Thêm exception cho file này
   → Antivirus đôi khi nhầm lẫn file PyInstaller
   → File hoàn toàn an toàn!

4. Đợi 2-3 giây

5. Browser tự động mở tại: http://localhost:5000

6. Bắt đầu sử dụng!

✅ Tối ưu cho tiếng Việt

TROUBLESHOOTING:

Q: Port 5000 đã được sử dụng?
A: Tắt app khác đang dùng port 5000
   Hoặc restart máy

Q: Không tự động mở browser?
A: Mở browser thủ công
   Truy cập: http://localhost:5000

Q: File .exe không chạy?
A: 1. Check antivirus
   2. Right click .exe → Properties → Unblock
   3. Chạy với quyền Administrator

Q: Lỗi "Missing dependencies"?
A: File .exe đã chứa tất cả dependencies
   Nếu vẫn lỗi, báo lại người gửi
